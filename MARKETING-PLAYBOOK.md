## Google Analytics (GA4) Setup

1. Go to [analytics.google.com](https://analytics.google.com)
2. Click "Start measuring" → Create an account for "Day Zero Solutions"
3. Create a property for "day-zero.com.au"
4. Choose "Web" as the platform
5. Enter URL: `day-zero.com.au` and stream name: "Day Zero Website"
6. Copy the Measurement ID (starts with `G-`)
7. Replace `G-XXXXXXXXXX` in `index.html` with your Measurement ID
8. Verify data is flowing in GA4 Real-time reports

### Google Search Console Setup
1. Go to [search.google.com/search-console](https://search.google.com/search-console)
2. Add property → URL prefix → `https://day-zero.com.au`
3. Verify via HTML tag or DNS record
4. Submit sitemap: `https://day-zero.com.au/sitemap.xml`
5. Monitor indexing status and search performance
